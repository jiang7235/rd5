##6.14学习总结##
###今日学习内容###
- git config：配置相关信息
- git clone：复制仓库
- git init：初始化仓库
- git add：添加更新内容到索引中
- git diff：比较内容
- git status：获取当前项目状况
- git commit：提交
- git branch：分支相关
- git checkout：切换分支
- git merge：合并分支
- git reset：恢复版本
- git log：查看日志
