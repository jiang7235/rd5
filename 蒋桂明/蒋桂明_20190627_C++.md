##6.27学习总结
###今日学习内容：

- lambda 表达式
- 常见基本算法
- 常用的STL算法
- set 算法

主要了解了lambda表达式的基本意义，学习编写了常见的几种STL算法，例如排序算法，遍历算法等。编写过程中有些麻烦；主要还是敲了这些最优的算法代码，这也给自己一个提示，以后自己工作时可以将想到的比较优秀的算法先存入STL中，然后当自己想要调用时候可以直接从库里调用而不需要重新编写代码，极大地减轻了程序编写的负担。
